# MVVM Movie Manager

Android-проект, демонстрирующий базовое использование архитектуры **MVVM** с **Room**, **LiveData**, **DataBinding** и **RecyclerView**.  

С помощью этого приложения можно:  
- Просматривать список жанров.  
- Просматривать фильмы по выбранному жанру.  
- Добавлять, редактировать и удалять фильмы.  

Приложение построено с упором на чистую архитектуру и современный Android-стек.

---

## Функционал

1. **Главный экран**  
   - Spinner для выбора жанра.  
   - RecyclerView с фильмами выбранного жанра.  
   - Возможность удалить фильм свайпом влево.  
   - FAB для добавления нового фильма.  

2. **Экран добавления/редактирования фильма**  
   - Ввод названия и описания фильма.  
   - Автоматическое сохранение изменений.  
   - Простая проверка на пустое название фильма.  

3. **База данных**  
   - Room Database с двумя таблицами: `genres_table` и `movies_table`.  
   - Изначально заполнена несколькими жанрами и фильмами
   - Связь `Movie → Genre` через foreign key с каскадным удалением.  

---

## Технологии

- **Kotlin**  
- **MVVM (ViewModel + Repository + LiveData)**  
- **Room** (Dao, Database, Entity)  
- **DataBinding**  
- **RecyclerView + Adapter**  
- **Coroutines** (для операций с БД)  
- **AndroidX**
